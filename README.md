# 易考在线考试系统

基于Vue+Electron的在线考试应用，包括用户端，教师端，和考试应用程序

教师端支持AI自动组卷，多种方式导入试卷，个性化分类题库，试卷，在线查看考生的摄像头，屏幕内容，实时对比考生人脸信息，实时发送消息提醒考生，实时警告考生作弊行为
